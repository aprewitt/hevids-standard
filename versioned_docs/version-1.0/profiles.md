# 4. Profiles

HEVIDS adopts a **Profiles** concept, distinguishing between:

- **Current Profile** — the set of HEVIDS Subcategories an organization can presently demonstrate evidence for, as found through self-assessment or third-party audit.
- **Target Profile** — the set of HEVIDS Subcategories an organization intends to satisfy within a defined planning horizon, given its risk tolerance, regulatory exposure, and the criticality of its AI deployments.

The gap between an organization's Current Profile and Target Profile constitutes its prioritized governance roadmap. HEVIDS does not mandate a single Target Profile for all organizations; a low-risk internal tool and a high-stakes hiring or lending system warrant different Target Profiles.
