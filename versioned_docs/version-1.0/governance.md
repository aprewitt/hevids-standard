# 6. Governance, Contribution, and Public Comment

HEVIDS is maintained as a living standard. Substantive changes are versioned and recorded in the [Changelog](./changelog). Public comment is open during the draft period preceding each numbered release.

To propose a change, report an ambiguity, or submit a public comment, please open an issue at [github.com/dradpa/hevids-standard/issues](https://github.com/dradpa/hevids-standard/issues).
